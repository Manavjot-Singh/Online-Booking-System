Online booking system for a fictitious airline

An online booking system for a fictitious airline, Albany Airlines, that allows users to search for flights, book tickets, and manage their bookings. The system is built using Django and provides a user-friendly interface for customers to interact with the airline's services. There are 5 main pages: Home, Search, Book, Details, and About. Customer can search for flights based on origin and destination, view flight details, and book tickets. The system also includes a booking lookup feature that allows customers to find their bookings using their reference number and are also able to cancel their tickets using same.The customers can also search for all their bookings using their email address. The system does not let customers book flights that are already fully booked or with departure dates and times that are not available.
